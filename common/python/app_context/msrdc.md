# gnome 2048 (ubuntu) Navigation Guide

**Standard Behaviors:**
- The app responds to standard GNOME desktop keyboard shortcuts.
- Navigation is primarily keyboard-driven.

**Navigation Paths:**
## Menus and Shortcuts
- **App Menu:** Super + A
- **Switch between windows:** Super + Tab (or hold Shift for reverse order)
- **Cycle through windows in the same application:** Alt + F6
- **Switch between workspaces:** Super + Page Up / Super + Page Down

## Keyboard Navigation
- **Use arrow keys** for general navigation.
- **Pop up command window:** Alt + F2

## Accessing Settings and Help
- **Settings:** Typically found in the app menu (Super + A), then look for 'Settings' or 'Preferences'.
- **Help/Support:** May be accessed via the app menu or by searching for 'Help' in the GNOME activities overview (Super key).

## Additional Tips
- The Super key (often the Windows key) opens the activities overview.
- For more shortcuts, refer to Ubuntu Linux keyboard shortcut guides.
