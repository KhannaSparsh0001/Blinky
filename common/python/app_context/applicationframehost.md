# Application Frame Host (applicationframehost.exe) - Settings

**Known Behaviors:**
- This application hosts various Windows settings.
- It utilizes standard Windows navigation and shortcuts.

**Navigation Paths:**
- Open Settings: Press `Win + I`
- Access Help and Support: Press `Win + F1`
- Clipboard History: `Win + V`
- Quick Link menu: `Win + X`
- Ease of Access menu: `Win + U`

**Common Shortcuts:**
- Maximize window: `Win + Up Arrow`
- Open Widgets: `Win + W`
- Menu navigation: `Alt + underlined letter of menu name`
- Clipboard settings: Settings app > System > Clipboard

**Tips:**
- Use `Win + Tab` to cycle through open windows.
- Press `Alt + Tab` to switch between open applications.
