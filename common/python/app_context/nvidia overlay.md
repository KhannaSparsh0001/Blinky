# NVIDIA GeForce Overlay

## Standard Behaviors
- The NVIDIA GeForce Overlay is a feature of the NVIDIA App and GeForce Experience.
- It provides in-game overlay functionality, including performance and latency monitoring, screen recording, and more.
- Default keybind to open/close overlay: Alt + Z (can be changed).

## Navigation Paths
### Opening the Overlay
- Press Alt + Z (default) or a custom keybind set in settings.

### Settings and Configuration
- Open the overlay.
- Click on the settings icon (usually represented by a gear).
- Navigate to "Settings" > "Keyboard Shortcuts" > "General" to change the open/close overlay shortcut.
- Navigate to "Settings" > "Features" to enable/disable the in-game overlay and performance statistics.

### Performance Statistics Overlay
- Press Alt + R to toggle statistics overlay on/off.
- Press Alt + Shift + R to cycle through different statistics.

### Troubleshooting and Support
- Consult the NVIDIA GeForce Experience or NVIDIA App documentation for help.
- Visit the NVIDIA GeForce Forums or Reddit community for user support and discussions.
